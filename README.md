# like-mouseimp
*[MouseImp](https://www.mouseimp.com/)*-like scrolling by dragging the mouse with the right button depressed.<br>
For Linux only.

Features:
- Vertical scroll: <kbd>RBtn</kbd> + Y-drag
- Horizontal scroll: <kbd>LeftShift</kbd> + <kbd>RBtn</kbd> + X-drag
